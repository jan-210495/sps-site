Place the Hs - Alhuda webfont files here. Suggested filenames:

- Hs-Alhuda-Regular.woff2
- Hs-Alhuda-Regular.woff
- Hs-Alhuda-Bold.woff2 (optional)
- Hs-Alhuda-Bold.woff (optional)

The stylesheet at assets/css/style.css includes @font-face rules that
reference these names and will automatically use the font when present.

If you use different filenames, update the src: URLs in the @font-face blocks.

Licensing: ensure you have rights to self-host this font.
